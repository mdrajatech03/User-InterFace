# 🔐 Modern Login & Registration Page

A lightweight, responsive, and user-friendly authentication interface. This project focuses on clean UI design and smooth interactivity, making it a perfect starting point for any web application's user gateway.

## 🚀 Key Features

* **Responsive Layout:** Fully optimized for mobile, tablet, and desktop screens using CSS Flexbox/Grid.
* **Smooth Transitions:** Interactive UI with elegant transitions between the Login and Sign-up forms.
* **Client-Side Validation:** Basic form handling to ensure data integrity before submission.
* **Clean Architecture:** Well-organized folder structure and commented code for easy customization.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure for better accessibility.
- **CSS3:** Custom styling, animations, and responsive design.
- **JavaScript:** Dynamic form switching and basic input logic.

## 📂 Project Structure

- `index.html` - The core structure of the application.
- `style.css` - Visual styling and layout management.
- `script.js` - Functional logic for UI interactivity.
- `login.jpg` - Featured background/brand asset.
- `icon.png` - Project favicon or branding logo.
